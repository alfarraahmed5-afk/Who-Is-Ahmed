# Ahmed AlFarra — Portfolio

> Personal portfolio website for Ahmed AlFarra, Urban Planner & GIS Spatial Analyst.  
> Live at **[alfarraahmed5-afk.github.io/Portfolio](https://alfarraahmed5-afk.github.io/Portfolio/)**

---

## About

Single-page portfolio showcasing urban planning projects, GIS work, professional experience, and academic background. Built as a static HTML file — no frameworks, no build step, no dependencies to install.

---

## Features

- Animated hero with live city-grid canvas
- GSAP scroll-triggered reveals and entrance animations
- Skill bars that animate on scroll
- GPA counter animation
- Custom cursor with lag effect
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll navigation
- Zero build tooling — one file, opens anywhere

---

## Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |

## Local Development

No build step needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/alfarraahmed5-afk/Portfolio.git

# Open directly in your browser
open Portfolio/index.html
```

Or use any local server if you prefer:

```bash
npx serve Portfolio
# → http://localhost:3000
```

---

## Deployment

Hosted on **GitHub Pages** from the `main` branch root.

Any push to `main` automatically redeploys within ~60 seconds.

```bash
git add index.html
git commit -m "update portfolio"
git push origin main
```
